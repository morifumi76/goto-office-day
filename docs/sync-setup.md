# 複数端末で色を共有する設定手順

別のPC・スマホで開いても出社日・帰省日の色を共有するための設定です。
色のデータを自分のGoogleスプレッドシートに保存し、ウィジェットがそこと通信します。

所要時間: 約10分。一度設定すれば以降は不要です。

---

## 1. スプレッドシートを作る

1. [Googleスプレッドシート](https://sheets.google.com) で空のシートを新規作成する。
2. 名前は何でもOK（例:「出社カレンダー保存先」）。

## 2. Apps Script を貼り付ける

1. スプレッドシートのメニューで **拡張機能 → Apps Script** を開く。
2. 最初から入っているコードをすべて消す。
3. このリポジトリの `docs/sync-apps-script.gs` の中身をすべてコピーして貼り付ける。
4. 💾（保存）を押す。

## 3. ウェブアプリとしてデプロイ（公開）する

1. 右上の **デプロイ → 新しいデプロイ** をクリック。
2. 歯車アイコン → **ウェブアプリ** を選ぶ。
3. 設定を以下にする:
   - **次のユーザーとして実行**: 自分
   - **アクセスできるユーザー**: **全員**
     - ※ ログインしていないウィジェットから読み書きするため「全員」が必要。
4. **デプロイ** を押す。
5. 「アクセスを承認」を求められたら、自分のGoogleアカウントで承認する。
   - 「このアプリは確認されていません」と出たら、**詳細 → （プロジェクト名）に移動 → 許可** で進める（自分が作ったスクリプトなので問題なし）。
6. 表示される **ウェブアプリのURL**（`https://script.google.com/macros/.../exec` で終わるもの）をコピーする。

## 4. ウィジェットにURLを貼る

1. `src/index.html` を開き、上のほうにあるこの行を探す:
   ```js
   const SYNC_URL = "";
   ```
2. コピーしたURLを貼り付ける:
   ```js
   const SYNC_URL = "https://script.google.com/macros/.../exec";
   ```
3. 保存して、GitHub に反映（push）する。

これで完了。どの端末で開いても同じ色が表示・保存されるようになります。

---

## 補足・注意

- **共有＝全員が編集できる**: ウィジェットを開ける人は誰でも色を変更できます。個人・社内利用なら問題ない範囲ですが、Notionページを社外に共有している場合は注意。
- **URLは空でも動く**: `SYNC_URL` が空のままなら、従来どおりその端末の中だけに保存されます（壊れません）。
- **データの実体**: スプレッドシートの「data」タブに、日付と種類(office/home/gray)が1行ずつ入ります。ここを直接編集してもOK。
- **コードを直したとき**: Apps Script を修正したら、**デプロイ → デプロイを管理 → 編集（鉛筆）→ バージョン「新バージョン」→ デプロイ** で反映する（URLは変わりません）。
- **同時編集**: 2台でほぼ同時に違う日を変更しても、1日単位で保存するので原則ぶつかりません。タブに戻ると自動で最新を取得します。
