# 出社カレンダー_指示書

> 出典: Notion「出社カレンダー_指示書」
> https://app.notion.com/p/38364d2aa0e880fa9f2fd1e2baf889f7

名古屋キャンパスへの「出社日」を塗って管理する、Notion埋め込み用のカレンダーウィジェットを作るための指示書。

## 📁 プロジェクト情報
| 項目 | 内容 |
|---|---|
| ローカルパス | `/Users/fumiyamorita/dev/gadget/goto-office-day` |
| GitHubリポジトリ名 | `goto-office-day`（小文字ケバブケース） |
| 公開方法 | GitHub Pages（mainブランチ） |
| 最終形 | Notionページに `/embed` で貼り付けて表示 |
| メイン成果物 | `src/` に置く単一HTMLファイル（例: `index.html`） |

## 🎯 目的・背景
- 在宅と名古屋・なごのキャンパスへの出社の2スタイルで働いている。
- 「この日は必ずキャンパスに行く」という日をカレンダー上でポチッと塗って管理したい。
- ぱっと見で「今月・来月・再来月の出社予定」が分かるダッシュボード的なウィジェットがほしい。

## ✅ 機能要件
| 項目 | 仕様 |
|---|---|
| 表示する月 | 今日を基準に今月・来月・再来月の3ヶ月を横並び表示（年またぎも自動で繰り上げ） |
| 週の始まり | 月曜始まり（月 火 水 木 金 土 日） |
| 出社日の操作 | 日付を1回押す → 黄色／もう1回押す → 色が消える（トグルのみ） |
| 選択の色 | 黄色の1色のみ |
| 保存 | 画面更新後も保持（`localStorage`、キー: `nagoya-shukkin-days`） |
| 週末・祝日 | 土日と祝日・休日はセル背景をうっすら赤（出社日の黄色が優先） |
| 今日の表示 | 今日の日付に青い枠線 |
| 出社日カウント | 各月のタイトル横に「出社◯日」を自動表示 |

## 🎨 デザイン仕様（色）

## 🛠 技術要件・制約
- 単一のHTMLファイルで完結（CSS・JSも同ファイル内）。
- フレームワーク不使用（素のHTML/CSS/JS）。ビルド工程なし。
- 外部ライブラリ・CDN読み込みなし（オフラインでも動く）。
- 保存は `localStorage`（キー: `nagoya-shukkin-days`）。
- Notionの埋め込み（iframe）内で動くこと。レスポンシブで、狭いときは3ヶ月が折り返す。
- コメントは日本語で、非エンジニアでも読めるように。
